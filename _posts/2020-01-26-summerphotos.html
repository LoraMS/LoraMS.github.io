About this project

Standard Web application using Node.js, Express and MongoDB with public part (accessible without authentication) and private part (available for registered users). This is an app for sharing your travels, tell where you've been, what you've done and display the pictures you've taken. Users can create publication or guide, like them and add them to their favorites, write comment to publications.
