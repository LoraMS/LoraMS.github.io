За този проект

Single Page приложение със следната функционалност: страница за вход в системата, потребителска кошницата, търсене на картина по заглавие или художник, филтриране на картини по стил / тема / техника, оценка на картина, коментар под картина.

Използвани технологии

  • jQuery
  • Handlebars
  • Less
  • Bootstrap
  • SystemJS
  • SammyJS
  • Moment
  • Firebase API, Database
  • Firebase API, Host
