About this project

Healthy Recipes is a Single Page Application created with Angular. The user can find different recipes, articles with healthy tips and nutrition information. The application can be used from non-registered users, registered users and has administration part.Registrated users can create new recipe or article, leave a comment, rate recipe and see own profile and collections.
