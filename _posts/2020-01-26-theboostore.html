About this project

The Bookstore is Single Page Application using React on front end, Node.js, Express on back end, MongoDB(Database/Authentication). The Web Application consists of the following sections: Home section, Books section, Events section, Authentication section, User Profile section, About section, ShoppingCart section. It has public part(accessible without authentication), private part(available for registered users) and administrative part(available for administrators only).
